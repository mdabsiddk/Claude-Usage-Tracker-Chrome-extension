## 🎯 বৈশিষ্ট্যসমূহ

### 📊 তিনটি প্রগতি ট্র্যাকার
Claude chat box এর নিচে রিয়েল-টাইম ব্যবহার পরিসংখ্যান দেখুন:

| ট্র্যাকার | বিবরণ | আইকন |
|---------|-------|-------|
| **⚡ Session** | ৫ ঘণ্টার সেশন লিমিট + কাউন্টডাউন | ⏱️ |
| **📅 আজকে** | দৈনিক সর্বোচ্চ ব্যবহার | 📆 |
| **🗓️ এই সপ্তাহ** | সাপ্তাহিক সর্বোচ্চ ব্যবহার | 📊 |

### 🎨 স্বয়ংক্রিয় রঙ সূচক

```
🟢 সবুজ      → সাধারণ ব্যবহার (0-70%)
🟡 হলুদ      → মাঝারি ব্যবহার (70%+)
🔴 লাল       → উচ্চ ব্যবহার (90%+) - পালস এনিমেশন সহ
```

### 🎛️ স্মার্ট প্ল্যান সিলেক্টর
Extension আইকনে ক্লিক করুন এবং:
- 📈 বিস্তারিত পরিসংখ্যান দেখুন
- 🔄 আপনার প্ল্যান নির্বাচন করুন (Free / Pro / Max)
- ⚙️ ব্যক্তিগত সীমা সেট করুন

## ⚙️ ইনস্টলেশন গাইড

### ধাপ-দ্বারা-ধাপ প্রক্রিয়া

#### **ধাপ ১:** ZIP ফাইল ডাউনলোড করুন
```
1. এই রিপোজিটরি ক্লোন করুন বা ZIP ফাইল ডাউনলোড করুন
2. ডাউনলোড বা ক্লোন করা ফাইলের ভিতরে থাকা “claude-usage-tracker.zip” ফাইলটি unzip করুন।
```

#### **ধাপ ২:** Chrome Extensions পেজে যান
```
🔗 URL: chrome://extensions
অথবা
1. Chrome খুলুন
2. ☰ অথবা 🚦 মেনু থেকে → “Extension” → Manage extensions এ ক্লিক করুন।
```

#### **ধাপ ৩:** Developer Mode সক্রিয় করুন
```
1. উপরে ডান কোণে "Developer mode" টগেল খুঁজুন
2. টগেলটিতে ক্লিক করে চালু করুন (🔵 নীল হবে)
```

#### **ধাপ ৪:** এক্সটেনশন লোড করুন
```
1. "Load unpacked" বাটনে ক্লিক করুন
2. unzip করা "claude-usage-tracker" ফোল্ডার সিলেক্ট করুন
3. ওপেন করুন ✅
```

#### **ধাপ ৫:** Claude.ai এ যান এবং উপভোগ করুন
```
1. 🌐 claude.ai খুলুন
2. Chat box এর নিচে progress bar দেখবেন
3. ✅ সম্পন্ন!
```

## ⚠️ গুরুত্বপূর্ণ তথ্য

### API লিমিটেশন
Anthropic একটি **পাবলিক API** প্রদান করে না ব্যবহার ট্র্যাকিংয়ের জন্য। এই কারণে:
- এই এক্সটেনশন **বার্তা সংখ্যা** ট্র্যাক করে
- **অনুমানিত টোকেন ব্যবহার** হিসাব করে (মডেল ভিত্তিতে)
- সঠিক সংখ্যা Claude ড্যাশবোর্ডের সাথে মিলতে পারে না

### নির্ভুলতা নোট
```
⚠️ এই পরিমাপ "অনুমানিত" - সঠিক হিসাব নয়
💡 প্রকৃত ব্যবহার জানতে Claude প্রফাইল দেখুন
🔗 claude.ai → Profile → Usage
```

### গোপনীয়তা সতর্কতা
- ✅ সমস্ত ডেটা **স্থানীয়ভাবে সংরক্ষিত** থাকে
- ✅ কোনো বাহ্যিক সার্ভারে পাঠানো হয় না
- ✅ সম্পূর্ণ **গোপনীয় এবং নিরাপদ**
