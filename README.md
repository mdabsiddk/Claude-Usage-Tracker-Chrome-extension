📦 ইনস্টল করার নিয়ম
ধাপ ১: ZIP ফাইলটি ডাউনলোড করে unzip করুন
ধাপ ২: Chrome এ যান → address bar এ টাইপ করুন: chrome://extensions
ধাপ ৩: উপরে ডান দিকে "Developer mode" toggle চালু করুন
ধাপ ৪: "Load unpacked" বাটনে ক্লিক করুন → unzip করা claude-usage-tracker ফোল্ডারটি সিলেক্ট করুন
ধাপ ৫: claude.ai খুলুন → chat box এর নিচে progress bar দেখতে পাবেন ✅

🎯 Features
Chat box এর নিচে তিনটি progress bar:

⚡ Session (৫ঘণ্টা) — কতক্ষণ পরে reset হবে countdown সহ
📅 আজকে — দৈনিক ব্যবহার
🗓️ এই সপ্তাহ — সাপ্তাহিক ব্যবহার

রঙ কোড: 🟢 সবুজ (স্বাভাবিক) → 🟡 হলুদ (৭০%+) → 🔴 লাল (৯০%+, pulse animation)
Popup: Extension icon ক্লিক করলে বিস্তারিত stats দেখবেন এবং Free/Pro/Max plan সিলেক্ট করতে পারবেন

⚠️ গুরুত্বপূর্ণ তথ্য
Anthropic কোনো public API দেয় না token tracking এর জন্য, তাই এই extension message count ট্র্যাক করে এবং অনুমানিত limits (Free: ~৩০/session, ~৬০/day) ব্যবহার করে। এটি একটি ভালো indicator কিন্তু Anthropic এর সার্ভারের actual limit demand অনুযায়ী পরিবর্তিত হয়।
